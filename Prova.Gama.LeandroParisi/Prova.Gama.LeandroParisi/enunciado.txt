Exercício Caixa Eletrônico
Fazer um programa do tipo Console que simule saque de um caixa 
eletrônico e sempre retorne o menor numero de notas possível.

Especificações:

1. As notas válidas são pode ser 100, 50, 20, 10, 5
2. As notas são infinitas
3. Todos os saques devem ser guardados em uma lista de Saques
4. O Saque deve conter a data e hora de quando foi realizado e o valor
5. Caso o saque não seja possível não registrar e informar a mensagem: “valor inválido”
6. No programa principal deve ser possível sacar de qualquer caixa eletrônico

DEVE-SE aplicar todos os conceitos passados em aula.

Formas de entrega:

Enviar a URL do repositorio no Github no Slack
Se caso não funcionar enviar um .zip pelo chat com o nome respectivo: NomeDoAluno.zip
Ex: FilipeCecconDeAlencar.zip
